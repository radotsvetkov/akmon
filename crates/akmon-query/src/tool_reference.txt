AKMON TOOL REFERENCE (concise)
==============================

All paths are sandbox-relative. Absolute paths and `..` are rejected.
Success shape: {"status":"success","content":"..."}
Error shape: {"status":"error","code":"snake_case","message":"..."}

TOOL: list_directory
Purpose: List files and folders at a path.
Input: path.

TOOL: read_file
Purpose: Read one UTF-8 file.
Input: path.

TOOL: write_file
Purpose: Create or replace a file atomically.
Input: path, content, optional dry_run=true (validate only, no write).

TOOL: edit
Purpose: Modify existing file by replacing one unique old_str.
Input: path, old_str, new_str, optional dry_run=true (validate only, no write).

TOOL: search
Purpose: Find regex text matches in files.
Input: pattern, optional path, optional file_pattern, optional case_sensitive.

TOOL: patch
Purpose: Apply unified diff to one or more files.
Input: patch, optional dry_run=true (validate only, no write).

TOOL: apply_patch
Purpose: Apply unified diff to exactly one file.
Input: file_path, patch, optional dry_run=true (validate only, no write).

TOOL: shell
Purpose: Run allowlisted shell command; cwd persists between calls.
Input: command.

TOOL: semantic_search
Purpose: Find relevant code by meaning.
Input: query, optional top_k.

TOOL: web_fetch
Purpose: Fetch a public HTTPS URL (SSRF protected).
Input: url.

TOOL: write_spec
Purpose: Save durable project spec loaded in later sessions.
Input: name, content.

TOOL: read_spec
Purpose: Read one spec or list specs.
Input: optional name.

TOOL: spawn_subagent
Purpose: Run isolated research agent and return summary.
Input: task.

TOOL: todo_write
Purpose: Update project task list stored in `.akmon/todos/current.json`.
Input: todos array with id, task, status.

TOOL: ask_followup
Purpose: Ask user a focused question and wait for answer.
Input: question, optional suggestions.

TOOL: memory_write
Purpose: Save project memory for future relevant retrieval.
Input: type, title, content.
